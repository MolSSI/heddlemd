# Writing a Simulation

> **Stub.** A simulation is fully specified by two files:
>
> - a TOML config (RNG seed, `n_steps`, `dt`, temperature, integrator mode,
>   masses, Lennard-Jones coefficients), and
> - an extended-XYZ init file (particle count, box, types, positions,
>   optional velocities).
>
> This chapter should walk through building both from scratch for a small
> custom system.
